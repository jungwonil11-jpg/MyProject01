수업 끝난 직후 실행. 강사 GitHub에서 최신 코드를 땡겨와 합치고, 내 GitHub에 push함.

## 강사 GitHub 주소
https://github.com/nohssam/2026-springboot01

## 실행 순서

1. `git remote -v` 로 upstream 등록 여부 확인
   - 없으면 자동 등록: `git remote add upstream https://github.com/nohssam/2026-springboot01.git`

2. `git status` 로 로컬 변경사항 확인
   - 있으면: `git stash push --include-untracked -m "강사싱크 전 임시저장"`
   - 없으면: stash 생략

3. 강사 repo fetch: `git fetch upstream`

4. upstream/main 머지: `git merge upstream/main`

5. stash 했으면 복원: `git stash pop`

6. 충돌 발생 시:
   - 충돌 파일 목록 알려줌
   - 강사 코드는 살리고, 내가 추가한 코드(새 파일, 새 기능)는 그대로 유지하는 방향으로 해결

7. 내 GitHub에 push: `git push origin main`

8. 완료 후 요약:
   - 강사가 추가/변경한 파일 목록
   - 충돌 해결 여부
   - push 완료 여부
