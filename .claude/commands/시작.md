다른 컴퓨터에서 자습 시작 전 실행. 내 GitHub 기준으로 로컬을 최신 상태로 맞춤.

## 실행 순서

1. `git status` 로 로컬 변경사항 확인
   - 있으면: `git stash push --include-untracked -m "시작 전 임시저장"`
   - 없으면: stash 생략

2. 내 GitHub pull: `git pull origin main`

3. stash 했으면 복원: `git stash pop`

4. 충돌 발생 시:
   - 충돌 파일 목록 알려줌
   - 내가 추가한 코드를 살리는 방향으로 해결

5. 완료 후 요약:
   - 새로 땡겨온 변경사항 목록
   - 현재 git status
